# 多 Agent 协同运营自动化系统

这是一个基于 [CrewAI](https://www.crewai.com/) 框架实现的多 Agent (智能体) 协同运营自动化系统示例。通过该系统，你可以看到多个不同角色的大模型 AI 智能体如何分工合作，自动化完成市场分析、文案创作、审核发布的完整运营流程。

## 🎯 项目解决的核心痛点

1. **告别单次 Prompt 的“AI 味”**：单靠一次对话让大模型写文案往往假大空且缺乏深度。本项目通过“多角色协作与批判”，让生成的内容更具人情味和网感。
2. **破解运营人力瓶颈**：传统的新媒体运营需要经历“热点调研 -> 策划 -> 撰稿 -> 审核 -> 发布”等长链路，耗时耗力。系统将整个流水线自动化，大幅提升内容产出效率。
3. **质量控制与品牌调性把控**：引入“总监审核”机制，自动拦截敏感词汇和低质量内容，确保每一次内容分发都符合商业战略。

## 🧠 核心逻辑流（长链推理与多Agent协作）

本系统采用了**多 Agent 协作 (Multi-Agent Collaboration)** 架构，通过清晰的角色边界构建了一条包含**长链推理 (Long-Chain Reasoning)** 的标准 SOP 工作流：

1. **任务输入**：人类给定一个宏观的运营目标（例如：推广一款新产品）。
2. **Agent 1 (深度推理与洞察)**：数据分析师接手任务，不急于写文案，而是先进行第一层逻辑推理——剖析目标受众、挖掘核心痛点、提炼出最容易引发共鸣的话题方向。
3. **Agent 2 (创意生成与约束执行)**：内容运营官接收分析师结构化的“洞察简报”，执行第二层逻辑——在严格遵循“小红书文风、Emoji 运用、话题标签”等设定下，将洞察转化为高转化率的种草文案。
4. **Agent 3 (批判性思维与决策)**：运营总监进行最后一层把关——运用批判性思维评估文案的商业价值与合规性。如果质量不达标，总监会指出问题（支持任务委派/Delegation，打回修改）；若通过，则补充最佳发布策略并输出最终定稿。

整个过程通过 `Process.sequential`（顺序流水线）串联，前一个 Agent 的思考成果自动成为后一个 Agent 的上下文，形成了一个**“调研 -> 创作 -> 审校”的严密闭环逻辑链**。

## 👥 角色分工

1. **资深数据与趋势分析师 (Data Analyst)**：负责根据主题提取用户痛点和市场热门话题方向。
2. **金牌内容运营官 (Content Creator)**：基于分析师的洞察，使用“网感”语言撰写爆款种草文案（支持 Emoji、热搜 Hashtags 等）。
3. **首席运营总监 (Operations Manager)**：负责对输出文案进行质量和合规性审核，最终拍板并提供渠道投放策略。

## 环境要求

- Python 3.10 或以上版本

## 安装与配置步骤

1. **安装依赖**
   在项目根目录下打开终端，执行以下命令安装所需库：
   ```bash
   pip install -r requirements.txt
   ```

2. **配置环境变量**
   - 复制 `.env.example` 文件，重命名为 `.env`。
   - 打开 `.env` 文件，填入你的大模型 API Key（`OPENAI_API_KEY`）。
   - 如果你想使用**国内大模型**（例如 DeepSeek、智谱 GLM、通义千问等，只要兼容 OpenAI API 格式即可），请在 `.env` 中取消注释并修改 `OPENAI_API_BASE` 和 `OPENAI_MODEL_NAME`。

   示例配置 (使用 DeepSeek API)：
   ```env
   OPENAI_API_KEY="sk-你的真实APIKey"
   OPENAI_API_BASE="https://api.deepseek.com/v1"
   OPENAI_MODEL_NAME="deepseek-chat"
   ```

## 运行方式

在终端执行：
```bash
python main.py
```

你将能在控制台中看到：
1. **分析师**首先工作并产出数据简报；
2. 随后交接给**内容运营**产出小红书图文；
3. 最后**总监**介入进行审核，输出最终定稿以及投放建议。

## 进阶玩法扩展

该系统架构具有极高的扩展性，你可以：
- 在 `tasks.py` 中分配**搜索工具** (Tools)，让分析师能实时联网抓取微博/知乎热搜榜。
- 增加新的角色，比如：**美工 Agent**（调用 Midjourney/DALL-E API 自动生成配图）。
- 在最后一步使用 **Selenium/Playwright** 自动将审核过的文案发布到各大平台。
